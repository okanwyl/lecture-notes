q. Describe the steps involved in data mining when viewed as a process of knowledge discovery.
a. 
1. Data cleaning : a process that removes and transforms noise and inconsistent data
2. Data integration : where multiple data sources may be combined
3. Data selection : Where data relevant to the analysis task are retrieved from the database.
4. Data transformation : where data are transformed or consolidated into forms appropriate for mining
5. Pattern evaluation : a process that identifies the truly interesting pattern representing knowledge 
based on some interestingness measures
6. Knowledge presentation : where visualization and knowledege representatiton techniques are used to present
the mined knowledge to the user.

q. How is a data warehouse is different from a database?
a. 

Data warehouse is a repository of information collected from multiple sources, over a history of time stored under
a unified schema, and used for data analysis and decision support, wheras a database is a collection of interrelated
data that represents the current status of the stored data.
